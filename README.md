# Real-time Asset Yield Management Audit Reporting Tool

This application is a web-based asset management and audit reporting system designed to improve organizational control over both equipment (e.g., laptops, printers) and consumable (e.g., tissue, paper, ink) resources.

It provides a centralized platform to:

1. Register and manage assets
2. Monitor asset usage and consumption
3. Conduct structured audit logging
4. Generate reports and summaries
5. Track resource utilization over time
